# Proyecto Iris

**Iris** es un sistema redirector de mensajes estructurados para entornos académicos, desarrollado como práctica de la asignatura de Estructuras de Datos y Algoritmos (EDA).

## Resumen del Proyecto
El objetivo es limpiar el "ruido" de los correos masivos académicos. El sistema:
1.  Recibe correos y busca la asignatura en un **Árbol Binario**.
2.  Localiza a los alumnos en una **Lista de Nodos**.
3.  Encola los mensajes en el buzón del alumno (**Cola**).
4.  Muestra un resumen ordenado por fecha inversa (**Pila**).
