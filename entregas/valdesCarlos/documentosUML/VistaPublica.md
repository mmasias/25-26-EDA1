# Vista Pública Reto-005

## Clase Pedido

- Representa un pedido individual.

- Guarda tipo, tiempo, llegada y estado del pedido.

- Permite reducir su tiempo minuto a minuto.

- Calcula su tiempo de espera.

## Clase ColaPedidos

- Representa la estructura de datos:

- Almacena todos los pedidos.

- Permite insertar nuevos pedidos.

- Permite extraer el pedido con menor tiempo.

- Cuenta el número de comparaciones realizadas.

## Clase Simulacion

- Contiene el método main.

- Ejecuta toda la simulación.

- Controla el avance del tiempo.

- Muestra el resumen final.
