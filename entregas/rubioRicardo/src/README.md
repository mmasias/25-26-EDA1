# Parcial: Sustitución de una Lista por una Lista Basada en Arrays 

Luis Felipe Brenes, Leslie Romero, Ricardo Rubio, Bernardo Zablah

## Descripción general

Nosotros teníamos un código que simulaba un array utilizando una lista. El objetivo es reemplazar la lista utilizada en nuestro código por la lista simulada por arrays del código del otro grupo. Es decir, tener un array simulado por una lista, que a su vez está simulada con array.

Extraemos el código del otro grupo, añadiéndolo como si fuera una librería. Eliminamos nuestra clase Nodo() y Lista(), para a continuación usar las 3 clases del otro grupo.
