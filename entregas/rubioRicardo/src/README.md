Propuestas de Solución
Aqui hay dos propuestas:

1. La Inicial (Lista Enlazada):
La inicial que fue usar lo que llevamos usando la mayoria del curso que es una Lista enlazada. A esta le toca recorrer toda la lista, lo cual no es muy eficiente pero funciona.

2. La Óptima (Min-Heap):
Hablando con mis compañeros de fila pensamos en lo último que habíamos visto que es un árbol. Buscando la solucion a esta ineficiencia en inteligencias artificiales, nos guía a la direccion correcta, que es un arbol binario llamado Montículo Mínimo (Min-Heap).

Es perfecto para esto ya que este tipo de arbol solo sigue una simple regla:

Cualquier nodo "padre" es siempre menor que sus dos nodos "hijos".

Esto hace que el elemento mas pequeño siempre esta en la raiz. Para lograr esto usa un array y usa formulas matematicas para convertirlo en arbol.
