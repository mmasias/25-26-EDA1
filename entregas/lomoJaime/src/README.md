# Simulación RCCCF – Restaurante de Cocina Optimizada

## Características

- **Simulación minuto a minuto** durante 120 minutos (jornada completa).
- **No se atienden en orden de llegada**, sino **por menor tiempo de preparación**
- **Llegada aleatoria de pedidos**: 40% de probabilidad por minuto.
- **5 tipos de platos**, con tiempos de preparación definidos:
  - Bebida: 1–2 min
  - Café: 2–3 min
  - Colacao: 2–4 min
  - Bocadillo: 3–5 min
  - Ensalada: 5–8 min
- **Un cocinero** que siempre elige el pedido pendiente con **menor tiempo de preparación**.
- **Métricas finales**:
  - Número de pedidos atendidos y pendientes
  - Tiempo total y medio de espera
  - Número de comparaciones realizadas
